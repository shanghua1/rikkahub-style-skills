# RikkaHub Style Skills

This repository contains a RikkaHub-compatible skill for roleplay style control.

## Install

In RikkaHub, install from this GitHub repository URL:

`https://github.com/shanghua1/rikkahub-style-skills`

The repository root contains `SKILL.md` for RikkaHub import.

The same skill is also mirrored at:

`.agents/skills/plain-rp-dialogue-guard/SKILL.md`

Use this skill to keep Chinese roleplay dialogue natural and prevent professional-domain jargon drift, especially computer, networking, information-security, system, protocol, optics, electronics, engineering, and information-theory metaphors.
